# ເອກະສານອ້າງອີງ

[1] https://react.dev ເປັນເອກະສານຫຼັກຂອງ React ທີ່ໃຊ້ໃນການສ້າງ UI ຂອງລະບົບ ໂດຍໃຊ້ Component-based Architecture.

[2] https://flask.palletsprojects.com ເປັນເອກະສານຫຼັກຂອງ Flask Framework ທີ່ໃຊ້ເປັນ Backend ສຳລັບຮັບ-ສົ່ງຂໍ້ມູນລະຫວ່າງ Frontend ແລະ Database.

[3] https://www.sqlite.org ເປັນເອກະສານຂອງ SQLite ທີ່ໃຊ້ເປັນ Database ເກັບຂໍ້ມູນຜູ້ໃຊ້, ຮ້ານ, ແລະ ຜົນການແນະນຳທັງໝົດຂອງລະບົບ.

[4] https://ai.google.dev ເປັນ Platform ຂອງ Google ທີ່ໃຊ້ Gemini AI Model ໃນການສ້າງຄຳແນະນຳອຸປະກອນໃຫ້ຜູ້ໃຊ້ ໂດຍເຊື່ອມຕໍ່ຜ່ານ API.

[5] https://ai.google.dev/gemini-api/docs ເປັນເອກະສານ API ຂອງ Gemini ທີ່ໃຊ້ອ້າງອີງໃນການ implement Google Search Grounding ແລະ Streaming Output.

[6] https://developers.google.com/custom-search ເປັນ API ຂອງ Google ທີ່ໃຊ້ຄົ້ນຫາລາຄາສິນຄ້າ ແລະ ຮູບສິນຄ້າ ຈາກເວັບໄຊ retail ຕ່າງໆ.

[7] https://open.er-api.com ເປັນ API ດຶງ Exchange Rate ທີ່ໃຊ້ convert ລາຄາ USD/LAK ແລະ THB/LAK ແບບ Real-time ໃນລະບົບ.

[8] https://pyjwt.readthedocs.io ເປັນເອກະສານຂອງ PyJWT ທີ່ໃຊ້ສ້າງ ແລະ ກວດສອບ JWT Token ສຳລັບລະບົບ Authentication ຂອງທຸກ Role.

[9] https://werkzeug.palletsprojects.com ເປັນ Library ທີ່ໃຊ້ Hash ແລະ ກວດສອບ Password ຂອງຜູ້ໃຊ້ດ້ວຍ `generate_password_hash` ແລະ `check_password_hash`.

[10] https://axios-http.com ເປັນ HTTP Client Library ທີ່ React ໃຊ້ຕິດຕໍ່ Flask API ໃນທຸກ Request ຂອງລະບົບ.

[11] https://lucide.dev ເປັນ Icon Library ທີ່ໃຊ້ສະແດງ Icon ຕ່າງໆໃນ UI ຂອງລະບົບ ເຊັ່ນ: BarChart, Store, ShieldCheck.

[12] https://vitejs.dev ເປັນ Build Tool ທີ່ໃຊ້ Run ແລະ Build React Application ຂອງລະບົບ.

[13] https://developer.mozilla.org/en-US/docs/Web/API/Server-sent_events ເປັນ Standard ຂອງ Server-Sent Events (SSE) ທີ່ໃຊ້ Stream ຜົນ AI ກັບ User ແບບ Real-time.

[14] https://jwt.io ເປັນເວັບໄຊ Reference ຂອງ JWT Standard ທີ່ໃຊ້ອ້າງອີງໂຄງສ້າງ Token Authentication ຂອງລະບົບ.

[15] https://restfulapi.net ເປັນເອກະສານ Reference ຂອງ REST API Architecture ທີ່ Flask Backend ຂອງລະບົບໃຊ້ໃນການອອກແບບ Endpoint.

[16] https://www.python.org ເປັນ Official Site ຂອງ Python ທີ່ໃຊ້ເປັນພາສາ Programming ຫຼັກ ໃນການພັດທະນາ Backend ຂອງລະບົບ.

[17] https://nodejs.org ເປັນ JavaScript Runtime ທີ່ຕ້ອງການໃນການ Run React ແລະ Vite Development Server.

[18] https://flask-cors.readthedocs.io ເປັນ Extension ຂອງ Flask ທີ່ໃຊ້ຈັດການ CORS ເພື່ອອະນຸຍາດໃຫ້ React (Port 3000) ຄຸຍກັບ Flask (Port 5000) ໄດ້.
